# Ubiquiti Assignment
Deployed [app](https://ubiquiti-nine.vercel.app/)

### Run app locally
(_Recommend to use node 20 and Chrome_)
#### Step 1
```bash
npm install
npm run dev
```
#### Step 2
Open [http://localhost:5173](http://localhost:5173) with your browser<br/><br/>

### Run unit test
```bash
npm run test
```

### Run e2e test
```bash
npm run cy
```
